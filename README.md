# Screen Finder

A responsive web application that allows users to browse and filter movie and TV show content.

## Features
- Search by title
- Filter by type (movie / series)
- Sort by release year
- Responsive mobile navigation

## Tech Stack
- HTML
- CSS
- JavaScript
- Font Awesome

## How to Run
1. Clone the repo
2. Open `index.html` in a browser

## Status
Some features are still in progress (placeholders included).
